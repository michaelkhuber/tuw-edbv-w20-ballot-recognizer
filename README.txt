Für das Verwenden unserer Programmes ist es empfohlen, die von uns erstellte GUI verwenden.
Hierfür muss die Datei "gui.mlapp" geöffnet, und daraufhin mittels des grünen "Run" - Buttons ausgeführt werden.

In unserer GUI kann dann von oben eines unserer Beispielbilder ausgewählt werden, oder alternativ kann eines vom Ordner geladen werden.
Dann kann mittels des Buttons "Run Pipeline" die Kreuzerkennung des Wahlzettels gestartet werden. Es öffnen sich hierbei zwei Figuren, 
eine große Figur, in der all unsere Transformations-Schritte dargestellt werden, und eine kleine Figur, in der die Kreiserkennung dargestellt wird.
Die gesamte Pipeline dauert etwa 20-40s auf guten Rechnern, und der Timer "Execution Time" updated sich währenddessen nicht, sondern
zeigt erst am Schluss die gebrauchte Zeit an.

Nach Abarbeiten der Pipeline wird in der Result-Box die vom Programm erkannte Kreuze aufgelistet, und ob diese valide oder nicht valide sind.
